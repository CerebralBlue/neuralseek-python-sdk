## execute

This endpoint takes an input object with a user question, context and options and returns a response object

### Example Usage

```python
from neuralseek import Neuralseek
import os

with Neuralseek(
    api_key=os.getenv("NEURALSEEK_API_KEY", ""),
) as s:
    res = s.seek.execute()

    if res is not None:
        # handle response
        pass

```

## stream

This endpoint takes an input object with a user question, context and options and returns a response object

### Example Usage

```python
from neuralseek import Neuralseek
import os

with Neuralseek(
    api_key=os.getenv("NEURALSEEK_API_KEY", ""),
) as s:
    res = s.seek.stream()

    if res is not None:
        with res as event_stream:
            for event in event_stream:
                # handle event
                print(event, flush=True)

```
