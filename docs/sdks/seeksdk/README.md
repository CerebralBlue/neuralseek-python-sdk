# SeekSDK
(*seek*)

## Overview

### Available Operations

* [execute](#execute) - Seek an answer from NeuralSeek

## execute

This endpoint takes an input object with a user question, context and options and returns a response object

### Example Usage

```python
from neuralseek import Neuralseek
import os

with Neuralseek(
    api_key=os.getenv("NEURALSEEK_API_KEY", ""),
) as n_client:

    res = n_client.seek.execute()

    # Handle response
    print(res)

```
